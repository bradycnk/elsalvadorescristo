# elsalvadorescristo
Organización sin fines de lucro que ayuda a conectar las iglesias de todo el planeta tierra y a personas dispuestas a aportar su granito de arena a comunidades vulnerables, discapacitados, ancianos, niños y niñas que necesiten de nuestra ayuda. Todo esto siguiendo el legado de Jesucristo: ama a tu prójimo
